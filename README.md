# HermesX

**本地优先的 AI 工作中枢，让模型不止于回答，而是真正把事情做完。**

HermesX 理解你的目标，接住散落的上下文，自主拆解任务、调度工具、读写文件，并把每一步推进透明地呈现在你眼前，让灵感、资料与成果在同一个空间里自然流动。

你只需说出想要抵达的结果，它便在对话、文件与任务之间建立秩序，把复杂工作收敛为可追踪、可掌控、可交付的过程。你负责判断方向，它负责推进落地；最终交付的不只是一段答案，而是一份可打开、可定位、可复盘的成品。数据与执行皆在本地完成，安全与掌控，始终在你这一侧。

## 版本日志

### v1.0.0 · 正式公测版（2026-06-21）

从 0.x 一路打磨到 1.0.0，HermesX 正式开启公测。这一版我们把重心放在「让 Agent 更自主、更安全」上：

- 子 Agent 调度：自动拆解复杂任务，派生子 Agent 并行推进
- 浏览器标签页操控：定位 CSS/Role/Text 元素，自动等待渲染就绪，截图即分析
- 持久 Shell 会话：跨命令继承目录与环境变量
- 项目上下文自动感知：读取 HERMESX.md 注入指令，Agent 更快进入状态
- 灾难防护：关键删除操作可撤销，误删有后悔药
- 安装包完整性校验 + 联网安全防护
- MCP 生态接入：GitHub / PostgreSQL 等外部工具即插即用

欢迎体验并把你的想法告诉我们，每一条反馈都会让正式版更好。
